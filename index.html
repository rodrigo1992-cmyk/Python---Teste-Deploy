Teste Deploy - GitHub Pages

Esta é uma página de exemplo para testar o deploy no GitHub Pages.

Como usar

Este repositório foi preparado para publicar a pasta raiz no GitHub Pages via GitHub Actions. Faça push para a branch main e a Action irá publicar automaticamente.

URL pública esperada

Após o primeiro deploy, a URL normalmente será:

https://<seu-usuario>.github.io/<nome-do-repositorio>/

Substitua <seu-usuario> e <nome-do-repositorio> conforme o repositório remoto.

Gerado automaticamente para testes de GitHub Pages.
